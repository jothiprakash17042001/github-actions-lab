# GitHub Actions Lab

## Setup

1. Create or activate your virtual environment:

```powershell
python -m venv .venv
.venv\Scripts\Activate.ps1
```

2. Install dependencies:

```powershell
python -m pip install -r requirements.txt
```

## Run the API

```powershell
python app.py
```

Then visit: http://127.0.0.1:5000/

## Run tests

```powershell
pytest
```
